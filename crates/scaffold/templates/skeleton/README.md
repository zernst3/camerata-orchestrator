## Develop

```bash
dx serve --platform web
```

Opens the fullstack dev server (SSR + hot reload). The native server binds
`localhost:8080` unless `PORT` is set.

## Quality gates

```bash
cargo fmt --all -- --check
cargo clippy --all-targets -- -D warnings
cargo check
cargo test
```

## Build

```bash
dx bundle --platform web --release
```

## Governance

This project ships with `CONVENTIONS.md` (structured/mechanical rules) and
`AGENTS.md` (prose-enforcement directives) adopted from the Camerata principle
library, plus `.github/workflows/ci.yml` which runs the build/test gate and the
governance check declared there. See those files before making structural changes.

## Deploy

A minimal Terraform module for a compute-only Azure Container App lives under
`terraform/`. No database is provisioned (this skeleton has none); see
`terraform/variables.tf` for the inputs it needs.
