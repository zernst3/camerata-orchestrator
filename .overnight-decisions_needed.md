## 2. iframe sandbox in the Mockup window (Queue Item 6)
**Decided**: `<iframe srcdoc>` without a Dioxus `sandbox` attribute (attribute not
exposed in the Dioxus element set; would need a raw `dangerous_inner_html` workaround).
**Risk**: The iframe runs with parent origin — XSS from AI-generated HTML is possible
if the AI produces malicious output. For local desktop use this is low-risk; for any
web-hosted deployment it needs tightening.
**Action needed**: If you host this as a web app, add sandbox via the
`additional_attributes` Dioxus escape hatch before shipping.
